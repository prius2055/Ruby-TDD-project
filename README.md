## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

- git clone https://github.com/prius2055/Ruby-TDD-project.git
- install ruby on your machine
 

### Prerequisites

In order to run this project you need to install ruby on your machine

### Setup

- install ruby on your machine

Clone this repository to your desired folder: https://github.com/prius2055/Ruby-TDD-project.git
 

### Usage

To use this project run the command `ruby person.rb` on the terminal.

### Run tests

run command `rubocop --auto-correct`
- test is written using RSpec
- commands to run: 
  - cd spec
  - rspec solver_spec.rb


### Deployment

You can deploy this project using gh-pages
